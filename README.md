# Customer Churn Retention Capstone

This project analyzes customer churn for a telecommunications company using machine learning.

## Tools Used
- Python
- Pandas
- Seaborn
- Scikit-learn
- Logistic Regression

## Key Results
- Model Accuracy: 78.9%
- Main churn driver: Fiber Optic Internet Service

## Files
- `notebooks/` – analysis and modeling
- `data_raw/` – dataset
- `outputs/` – visualizations
- `report/` – final project report
